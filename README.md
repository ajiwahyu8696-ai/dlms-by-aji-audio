# 🎛️ AJI AUDIO LUMAJANG - DLMS HOREG PRO v5.6.4

Aplikasi simulator manajemen audio berbasis web (HTML5 Web Audio API) yang dirancang khusus untuk kebutuhan tuning Sound System Horeg.

## 🚀 Fitur Utama (Full Restored)
* **DSP Sub-Bass Engine**: Mode *Low Cut (40Hz)*, *Super Bass*, dan *Flat/Off*.
* **Matrix Crossover**: Filter HPF & LPF dengan pilihan slope Linkwitz-Riley & Butterworth hingga 48dB.
* **15-Band Equalizer Matrix**: Pemisah frekuensi presisi tinggi untuk Low (Ch 1) dan High (Ch 2).
* **🔒 Horeg Peak Limiter**: Sistem proteksi speaker dengan pengaturan manual *Threshold*, *Attack*, dan *Release* (Pro Feature).
* **Echo Feedback & Time Delay**: Efek pantulan audio spasial pada Channel 2 untuk Mid/High.
* **Internal Test Tone Generator**: Pembangkit sinyal acuan berupa *Pink Noise* dan *Sine Wave* untuk kalibrasi frekuensi lapangan.
* **Firebase Cloud Integration**: Validasi lisensi *Pro Key* secara *online* dan fitur simpan/muat preset langsung dari *cloud database*.
